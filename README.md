## 🛠️ Technical Stack
- **Frontend:** HTML5, CSS3 (Glassmorphism & Matrix Rain Effect)

- **COMMUNITY:** A decryption-themed comment system for feedback.

- `script.js` - Matrix rain logic and report management.
